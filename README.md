# Computing 2 Coursework Submission.
**CID**: [02584835]

This is Orbito! A fun boardgame where each go a player can move one of their opponents pieces to an adjacent square if they want to before placing a piece in an empty space. The pieces then rotate anticlockwise by one slot (in the physical game this is done mechanically). After the board rotates, if either player has 4 in a row, they have won. Enjoy!

AI Declaration:
